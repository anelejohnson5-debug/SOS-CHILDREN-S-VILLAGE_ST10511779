Website for SOS Children's Village Ennerdale.
This website is created for the Sos Children's Village in ennerdale, the current website lacks a huge amount of information thus hindering it's influence on the public.
The aim for this website is for the community of ennerdale to be able to interact with the SOS family actively, raise awareness and give the ennerdale community and the rest of the public the much needed information.
The website will be created using, html,css and js.
The current problems i faced with creating the website thus far was having to use alternative methods to create and code for the website because my virtual machine would've ran out of time if i had to code directly from it, basically the main problem was time and struggling with fixing certain code and remembering certain aspects i might have forgotten.
Aditional features of the website will include clear navigation, updated content about the village, and sections that highlight events, initiatives, and ways the public can contribute or get involved.
The goal is to create a platform that is both informative and engaging for users of all ages.
Through overcoming these challenges, the project has also provided an opportunity to strengthen problem-solving skills, improve coding knowledge, and gain practical experience in web development.
Future improvements may include adding more interactive features, integrating a content management system, and continuously updating the site to better serve the needs of the community.
